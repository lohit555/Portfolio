My Projects

SafeGuard - AI Hazard Detection Web App

An AI-powered hazard detection web app built at MacHacks 2026. Users can upload images and the app uses Google Gemini to detect potential safety hazards in real time, supporting up to 8 concurrent images with automatic resizing.

Tech Stack: Node.js, Vercel Serverless, Google Gemini API, React, Vite

Catan - Board Game Simulator with AI Agent

A Java-based board game simulator featuring a rule-based AI agent that makes intelligent decisions at runtime. Built as a team project with full UML design and test coverage.

Tech Stack: Java, JUnit, Eclipse Papyrus, Strategy Design Pattern

GeoShield - Real-Time Disaster Response Web App

A real-time disaster response web app built at MEC 2025. Users can submit help requests and view safe zones, with all updates instantly reflected across connected users via Socket.io.

Tech Stack: Node.js, Express, Socket.io, REST APIs

WorDex - Command-Line Wordle Game

A C-based command-line Wordle game with persistent game history, a leaderboard system, and color-coded terminal output for a polished playing experience.

Tech Stack: C, File I/O, Command-Line
