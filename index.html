My Projects

SafeGuard - AI Hazard Detection Web App

SafeGuard is an AI-powered workplace hazard detection platform developed at MacHacks 2026 that analyzes workplace photos to identify safety risks and generate severity-ranked hazards, scene descriptions, and prioritized safety recommendations. Built with Node.js, Vercel Serverless Functions, and the Google Gemini API, it supports multimodal analysis across multiple images and helps users quickly assess and address workplace safety concerns.

Catan - Board Game Simulator with AI Agent

Catan is a Java-based board game simulator inspired by Settlers of Catan, developed as a team project at McMaster University. The game models core mechanics such as resource management, settlements, cities, roads, trading, the Robber, and AI-driven gameplay. Built with Java, JUnit, and Eclipse Papyrus, the project features a rule-based computer player, UML-driven software design, and comprehensive testing to ensure accurate and reliable gameplay.

GeoShield - Real-Time Disaster Response Web App

GeoShield is a real-time disaster response web app developed at the McMaster Engineering Competition (MEC) 2025 that helps coordinators and responders manage help requests, track missions, and monitor safe zones across connected users. Built with Node.js, Express, Socket.io, and React with Google Maps API integration, it enables real-time coordination and live updates for emergency response scenarios through a scalable, event-driven architecture.

WorDex - Command-Line Wordle Game

WorDex is a terminal-based Wordle-style game developed as a team project at McMaster University featuring user authentication, persistent game history, global leaderboards, and player statistics across sessions. Built using C with Makefiles and file I/O, it delivers a fully functional command-line gaming experience with structured data persistence and ranking systems.
